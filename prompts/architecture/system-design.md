# Prompt mẫu cho Kiến trúc & Review (Architecture)

## 1. Thiết kế Hệ thống (System Design)

```
Hãy thiết kế hệ thống cho ứng dụng [loại ứng dụng]:

Yêu cầu chức năng: [liệt kê]
Yêu cầu phi chức năng:
- Số lượng người dùng: [số lượng]
- Độ trễ tối đa: [ms]

Hãy cung cấp:
1. Sơ đồ kiến trúc mức cao (Mermaid).
2. Các thành phần chính và nhiệm vụ của chúng.
3. Lựa chọn công nghệ (Database, Cache, Message Queue) kèm giải thích.
4. Chiến lược mở rộng (Scaling strategy).
```

---

## 2. Review Code Bảo mật (Security Review)

```
Hãy rà soát đoạn mã nguồn này để tìm các lỗ hổng bảo mật:
[Dán code ở đây]

Tập trung vào:
- Tấn công Injection (SQL, XSS).
- Lỗi xác thực và phân quyền.
- Lộ thông tin nhạy cảm.
- Kiểm tra đầu vào (Input validation).

Với mỗi vấn đề, hãy nêu rõ: Mức độ nghiêm trọng, Vị trí lỗi và Cách khắc phục.
```
